# 个人管家

一个纯前端、离线可用的个人日常管理网页应用。所有数据保存在浏览器本地存储（localStorage）中，无需后端服务，打开即用。

---

## 项目介绍

「个人管家」是一款围绕日历展开的个人管理工具，帮助用户记录每日待办事项、跟踪习惯打卡、回顾每周表现。设计初衷是简洁、轻量、隐私优先——数据完全留在本地，不依赖任何云服务。

应用采用单文件架构（`index.html`），内含 HTML、CSS 与 JavaScript，可直接在浏览器中打开运行，也支持通过静态服务器部署。

---

## 功能列表

### 日历与导航
- 月历视图：快速浏览整月安排，点击日期进入详情
- 日期标记：日历格子上以圆点提示当日有待办或习惯记录
- 悬停预览：鼠标停留在日期上，自动弹出当天待办与习惯概览
- 键盘快捷键：左右方向键切换月份，`Esc` 返回日历

### 待办清单
- 按日期管理任务，支持添加、完成、删除
- 重复任务：可设置为每天、工作日或每周重复
- 拖拽排序：长按任务项可调整先后顺序
- 操作撤销：删除任务后弹出提示，支持一键撤销

### 习惯打卡
- 默认提供护肤、喝水（8杯）、运动30分钟、维C水果四项习惯
- 支持打卡型（是否完成）与计数型（喝水杯数）两种模式
- 自动计算连续打卡天数
- 习惯名称与类型可自由编辑、增删

### 数据看板
- 今日待办完成进度
- 本月待办完成率
- 本月习惯打卡总数
- 最长连续打卡天数

### 周视图与统计
- 本周待办：横向表格展示本周每一天的待办完成状态
- 习惯完成率：环形图展示本周各习惯的达成比例
- 每日小结：一键生成当日文字总结

### 月度报告
- 待办完成率与习惯打卡率总览
- 最佳习惯与最需坚持的习惯对比
- 习惯打卡趋势折线图（支持多习惯同时展示）

### 数据管理
- 导出备份：一键下载 JSON 格式的完整数据
- 导入恢复：选择备份文件还原数据
- 清理数据：自动清理90天前的旧记录，释放存储空间
- 深色模式：支持浅色/深色主题切换

### 搜索
- 全局搜索待办内容，快速定位历史记录

## 使用说明

### 直接打开
在浏览器中直接打开 `index.html` 文件即可使用。为了获得最佳体验，建议通过本地服务器访问：

```bash
# Python 3
python -m http.server 8080

# 或使用 Node.js 的 npx
npx serve .
```

然后在浏览器中访问 `http://localhost:8080/index.html`。

### 基本操作
1. **查看日历**：首页展示当月日历，带圆点的日期表示有记录
2. **进入日期详情**：点击任意日期，进入该天的详情页面
3. **添加待办**：在详情页输入任务内容，选择是否重复，点击「添加」
4. **打卡习惯**：在习惯区域点击圆形按钮完成打卡，或点击加减按钮记录喝水杯数
5. **查看周报**：在详情页下滑可看到本周待办表格和习惯完成率环形图
6. **月度报告**：点击顶部 📊 按钮，查看整月统计与趋势图

### 数据备份
- 点击页面底部的「导出」按钮，可将所有数据下载为 JSON 文件
- 点击「导入」并选择之前导出的 JSON 文件，即可恢复数据
- 建议定期导出备份，防止浏览器清理数据导致丢失

### 快捷键
| 按键 | 作用 |
|------|------|
| `←` / `→` | 切换月份 |
| `Esc` | 返回日历 / 关闭弹窗 |
| `Ctrl + F` | 打开搜索 |
| `Enter` | 添加待办 |

## 技术说明

- **纯前端实现**：无后端依赖，无需安装任何环境
- **单文件部署**：所有代码集中在 `index.html` 中
- **本地存储**：使用 `localStorage` 持久化数据
- **响应式设计**：适配桌面与移动设备
- **打印友好**：详情页支持打印输出

---

## 浏览器兼容性

支持所有现代浏览器（Chrome、Edge、Firefox、Safari）。由于使用 `localStorage` 存储数据，建议在私密浏览模式之外使用，以确保数据正常保存。
